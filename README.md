## Sobre o projeto

O **Sublimus Burger** é um sistema de pedidos para hamburgueria desenvolvido com foco em **mobile-first**. O cliente acessa pelo celular, navega pelo cardápio, adiciona itens ao carrinho e finaliza o pedido direto pelo WhatsApp — sem precisar de app ou cadastro.

-----

## ✨ Funcionalidades

- 📂 Filtro de produtos por categoria (Duas Carnes, Frango, Combo, Bebidas)
- 🛒 Carrinho lateral com adição, remoção e controle de quantidade
- 💾 Carrinho salvo no `localStorage` — persiste ao recarregar a página
- 📲 Pedido enviado via WhatsApp com lista de itens e total
- 📱 Tab bar estilo app nativo no mobile
- 🖥️ Layout responsivo — funciona no celular e no desktop
- 🎨 Identidade visual consistente com tipografia e cores da marca

-----

##  Tecnologias utilizadas

- **HTML5** — estrutura semântica
- **CSS3** — mobile-first, Flexbox, media queries, animações
- **JavaScript** — manipulação do DOM, eventos, localStorage
- **Font Awesome** — ícones
- **Google Fonts** — Fraunces, Montserrat, Bebas Neue, Anton, Oswald

-----

##  O que aprendi com esse projeto

- Desenvolver pensando **mobile-first** desde o início
- Criar componentes interativos sem frameworks — só JS puro
- Usar `localStorage` pra persistir dados no navegador
- Estruturar CSS com media queries pra múltiplos dispositivos
- Integrar um sistema de pedidos com **WhatsApp API**
- Organizar um projeto real do zero até o deploy
